Project Updater

Update project configuration files automatically
Update All
Run all update scripts in one go. This will:
  • Update Welcome Page - Scan preset files and update welcome.html
  • Update Overlay List - Scan overlay files and update overlayPresets.js
./scripts/update-all

How to Use

  1. Click a button above to reveal the command
  2. Copy the command or open your terminal
  3. Run the command from your project root directory
  4. Or use VS Code: ⇧⌘P → Tasks: Run Task
